# @keepintouch
## 소개
에버노트, 노션 등등에 저장된 나만의 명언, 오늘했던 좋은 생각, 다음에 또 보고싶은 문장들을 저장<br> 저장된 문구들 중 하나를 매일 나에게 보내주는 텔레그램 봇<br> 
슬래시 커멘드 형태로 동작(인공지능 똑똑이 아님)<br>
책읽고 저장해둔 글귀를 에버노트에서 5년 후 발견한 개발자의 안타까움에서 시작됨(개인용)
## 개발환경
node.js
heroku
mysql
